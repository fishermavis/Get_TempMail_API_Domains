# Temp Mail Domain Fetcher

A lightweight Python utility to retrieve the currently available domains from the **Privatix Temp Mail API**. This script is useful for developers building automated testing suites or temporary email services.

---

## Features
* **Real-time Updates:** Fetches the latest active domains directly from the RapidAPI endpoint.
* **Simple Integration:** Uses the `requests` library for clean, readable HTTP calls.
* **Lightweight:** Minimal dependencies and easy to read.

## Prerequisites
Before running the script, ensure you have:
* **Python 3.x** installed.
* A **RapidAPI Account** and an API key for the [Temp Mail API](https://rapidapi.com/privatix/api/temp-mail/).
